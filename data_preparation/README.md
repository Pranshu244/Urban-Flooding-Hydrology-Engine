## Data Preparation

This folder contains the notebook used to generate the geospatial dataset for the flood prediction model.
The data is extracted using Google Earth Engine and processed into a format suitable for machine learning.

### Steps to Run

1. Create a Google Earth Engine account.
2. Authenticate Earth Engine in Python.
3. Run the notebook:

   Hydrology_Engine_GIS_Data_Preparation.ipynb

This notebook will process satellite imagery, rainfall data, and spatial layers to generate the training dataset.
